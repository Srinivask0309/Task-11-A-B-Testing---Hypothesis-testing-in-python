# Task 11 – A/B Testing using Python

## Objective
Perform hypothesis testing to evaluate the impact of a new version on conversion rate.

## Tools Used
- Python
- pandas
- numpy
- scipy

## Steps Performed
1. Dataset creation and group identification
2. Hypothesis definition
3. Conversion rate calculation
4. Two-sample t-test
5. P-value interpretation
6. Confidence interval calculation
7. Business recommendation

## Result
Test group shows a statistically significant improvement in conversion rate.
